# u18sh-preinstall

Preinstall Webserver for clean VPS/VDS on Ubuntu 18

- Nginx
- NPM, NVM, Yarn
- NodeJs v14
- PHP 7.4 (+FPM)
- CertBot

```
$ wget https://raw.githubusercontent.com/KOYU-Tech/u18sh-preinstall/main/preinstall-u18.sh
$ sudo chmod +x ./preinstall-u18.sh
$ sudo ./preinstall-u18.sh
```
