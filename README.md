# ws-preinstall-u18

Bash Script for a webserver preinstallation on a clean VPS/VDS with Ubuntu 18

- Nginx
- NPM, NVM, Yarn
- NodeJs v14
- PHP 7.4 (+FPM)
- CertBot

```
$ wget https://raw.githubusercontent.com/KOYU-Tech/ws-preinstall-u18/main/wsp-u18.sh
$ sudo chmod +x ./wsp-u18.sh
$ sudo ./wsp-u18.sh
```
